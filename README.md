# 🗺️ Prague District Classification using MLP

**Course:** Neural Networks | **Grade:** 15/15 🏆
**📄 [View Presentation PDF](./Sagova_Prague_District_Classification_MLP.pdf)**

## 📌 Overview
This repository hosts my final presentation on spatial classification using a Multi-Layer Perceptron (MLP). The project predicts Prague city districts based purely on the GPS coordinates of local cafés, serving as a study of neural network learning dynamics.

## 🧠 Technical Skills
* **Architecture:** Built an MLP to map complex, non-linear geographic boundaries directly from raw latitude/longitude data.
* **Optimization:** Analyzed and compared SGD, Momentum, and Adam optimizers for learning stability and convergence speed.
* **Robustness:** Performed weight pruning to demonstrate a distributed, non-overfitted internal network representation.

## 🎨 Soft Skills & Visual Communication
The included presentation was custom-designed in **Canva**. It highlights my ability to take complex mathematical concepts (like decision boundaries, cross-entropy loss, and network sparsity) and communicate them clearly and visually to any audience.
